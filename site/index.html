Pane

A round-trip UI channel between agents and humans.

An agent hands a human a rich interactive UI by URL, captures their interactions as structured data, and feeds it back — no GUI host app needed on either side.

The idea

Agents can already emit rich output. But the human's reply is still a text box. Pane closes the loop: the agent renders a UI — a form, a picker, a doc-review view, a dashboard — the human acts on it, and every interaction comes back as structured data. The human answers by using a UI, not by typing.

It matters most for agents that live outside a GUI host app: cron agents, Slack and Telegram bots, CI jobs, headless servers. The agent only makes outbound calls to the relay — no host app, no public address of its own.

Get started

  1. 1
    Install the CLI: 
    npm i -g @paneui/cli
  2. 2
    Register an agent key against the hosted relay: 
    pane register --url https://relay.paneui.com
  3. 3
    Hand a human a UI and wait for the answer: 
    pane create --artifact ./form.html --schema ./schema.json
pane watch ses_xxxx --type form.submitted

Teach an agent to use it

The pane skill (Agent Skills format) installs into any agent. For Claude Code, use the plugin marketplace; for others, the cross-agent skills tool: 

# Claude Code
/plugin marketplace add aerolalit/pane
/plugin install pane@pane

# Codex, Cursor, Copilot, Gemini CLI, …
npx skills add aerolalit/pane --skill pane
